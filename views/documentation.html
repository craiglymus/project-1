Welcome to Bug Bounty, your comprehensive guide to Bug Stuff.

Here you will all you need to know about the little critters that share the world with us.

Our bug library collects species information and connects it to Wikipedia for an easy to use, accessible experience. Our users create the content they need most by adding insects and families to our ever-growing database. The Bug Bounty API implements a RESTful interface that can be used for identification, search, and sharing.

URL: localhost:3000/api/insects

method: 'GET', path: '/api/insects', description: 'Returns all insects in the database'

Returns all insect objects in the database

URL: localhost:3000/api/insects/:id

method: 'GET', path: '/api/insects/:id', description: 'Returns an individual by id.'

Returns an individual insect by its ID so it can be individually manipulated

URL: localhost:3000/api/insects/species/:name

method: 'GET', path: '/api/insects/species/:name', description: 'Returns an individual insect by common name.'

Returns an insect object by common name, used in the search function. Accepts partial names and will return any entry that contains that string in the common name.

URL: localhost:3000/api/insects/:id

method: 'PUT', path: '/api/insects/:id', description: 'Edits an existing entry in the database'

Edits an existing entry in the database, can only edit common name, species name, and description.
